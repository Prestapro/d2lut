## Price Sources

- **Primary:** d2jsp forum (https://forums.d2jsp.org/forum.php?f=271)
- **Runes:** Static price config (rune_prices.yml)
- **Runewords:** Generated from rune component prices
- **Uniques/Sets:** Database observations
- **Magic items:** Affix combination estimation (gg_affixes.yml)

---

## Known Limitations

1. **Dynamic pricing not possible** - D2R localization is static text
2. **Refresh requires game exit** - File is locked during gameplay
3. **Anti-bot protection** - d2jsp Cloudflare blocks automated scraping
4. **Roll-sensitive items** - Cannot determine exact value without seeing rolls

## Technical Details

### Input CSV Format (ff05e402.csv)
```csv
Category,Name,Property,Affix Level,Level Req,Item Types,Rarity,Group
Magic,Stalwart,+33 to 40 Defense,32,24,large charm,magic only,101
Magic,Jeweler's,Socketed (4),55,47,"weapon, any shield, helm, armor",magic only,122
```

### Rune Price Config (rune_prices.yml)
```yaml
prices:
  "Jah": 50
  "Ber": 55
  "Sur": 25
  "Lo": 12
  "Ohm": 15
  ...
```

### Price Formatting
- Prices ≥ 1 FG: integer format (e.g., "50 FG")
- Prices < 1 FG: decimal format (e.g., "0.5 FG")
- Color tags: `ÿc1` for red (GG), `ÿc0` for reset
